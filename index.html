Visual Check for N=1..35

Inspect one displayed witness for each value of N. Each witness is a product-distinct rectangle A x B inside [N] x [N].

Problem setting and small example

Problem setting: For each N, we compute M(N). Here M(N)=max |A||B|, where A and B range over subsets of [N]={1,2,...,N}, and all products a*b, with a in A and b in B, must be distinct.

Small example: For N=6, both sets are chosen from {1,2,3,4,5,6}. Here we choose A={1,2,3,4} and B={1,5,6}, so |A||B|=4*3=12. The products are 1,2,3,4,5,10,15,20,6,12,18,24, all distinct.
